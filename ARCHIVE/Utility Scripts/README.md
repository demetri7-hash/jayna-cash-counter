# Utility Scripts Archive

This folder contains utility scripts and one-off tools that are not part of the main application.

## JavaScript Utilities

- `check-email-parser.js` - Email parsing validation script
- `check-table-exists.js` - Database table existence checker
- `generate-label-mapping-excel.js` - Excel generator for label mappings
- `read-label-mapping.js` - Label mapping reader/generator
- `update-label-names.js` - Label name update utility

## HTML Utilities

- `cleanup-duplicate-emails.html` - Tool to clean duplicate email entries
- `unarchive-all-votes.html` - Teacher Feast contest vote unarchive tool

## Why Archived?

These are standalone utility scripts used for one-time tasks, debugging, or data cleanup. They are not referenced by the main application and are not needed for normal operations.

**These scripts may require specific setup, credentials, or database states to function correctly.**
